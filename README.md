# Install Kafka, Debezium, Postgresql, jupyter, kafka-manager

1. Kubernetes Preparation
```
cd kubernetes
./install.sh
```
2. Helm install 
```
cd helm
./install.sh
```
3. Jupyter Notebook install
```
cd jupyter
./install.sh 
```
4. Postgresql install
```
cd postgresql
./install.sh 
```
5. Kafka, Kafka connect, debezium, KSQL DB, etc install
```
cd kafka
./install.sh 
```
6. Kafka-manager UI 
```
cd kafka-manager
./install.sh 
```
